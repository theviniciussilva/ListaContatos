# Objetivo
Desenvolvi uma aplicação prática e intuitiva para gerenciamento de contatos, demonstrando minha habilidade em criar soluções reais com Redux.

# Tecnologias utilizadas

* Html
* Css
* Typescript
* React
* Redux

# Funcionalidades

* Cadastrar Contatos:
Registre informações como nome, email e telefone, com validação que evita duplicidade de dados.

* Editar e Atualizar:
Altere as informações dos contatos de forma simples, mantendo os dados sempre atualizados.

* Remover com Facilidade:
Exclua contatos quando necessário, com o estado global se atualizando automaticamente.

# Como Executar
* 1 - Clone o repositório.
* 2 - Instale as dependências.
* 3 - Inicialize a aplicação.
